# 한영타 변환기 (영타 -> 한글)

### 실행파일

* 맥OS - en2ko.app   
* 윈도우 - en2ko.exe

## 빌드
### 빌드 환경
* python 3.X ()
* tkinter (Python3 기본 라이브러리)
* pyinstaller   

### 빌드 방법
```
git clone https://github.com/J-Mook/en2ko.git
```
&nbsp;   
클론된 폴더에서 터미널
```
bash make.sh
```
